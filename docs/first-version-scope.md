# 第一版範圍

## 產品目標

第一版不是完整教學平台，而是能向老師、mentor 或競賽評審展示核心差異的 Android 原型：偏鄉學生在英文學習卡關時，不再只收到錯誤結果，而是被平台引導到更小的任務，並在需要時把資訊交給真人陪伴者。

## 使用者

- 學生：國中階段、英文基礎薄弱、容易因錯題累積而中斷學習。
- 志工/老師：時間有限，需要快速知道學生真正卡在哪裡。

## 核心流程

1. 學生進入平台後先選擇心情狀態。
2. 系統依心情安排今日任務量。
3. 學生進入短任務，若答錯，AI 用白話提示。
4. 若連續答錯三次，系統標記為情緒/學習斷點。
5. 系統改派更小的修復任務。
6. 志工端可查看整理後的卡關摘要與建議陪伴語。

## 暫不處理

- 帳號登入。
- 真實 AI 串接。
- 後端資料庫。
- 真實課綱題庫。
- 推播通知。

## 成功標準

- 使用者能在手機上走完學生端與志工端流程。
- 觀眾能一眼理解「AI 即時緩衝 + 真人接力」的雙軌設計。
- 後續能根據 UI 回饋繼續擴充，不需要重寫專案。
