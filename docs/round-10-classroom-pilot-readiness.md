# Round 10 - Classroom Pilot Readiness

This round prepares English+ for a controlled classroom or presentation pilot.

## Added

- `ClassroomPilotContract` with schema version `10`.
- A 45-minute pilot plan with teacher, student, and volunteer moments.
- Consent packet requirements:
  - student learning data notice
  - AI support context notice
  - guardian or school consent
  - teacher/volunteer access explanation
  - opt-out path
- Success metrics focused on usefulness rather than scores:
  - student return willingness
  - teacher actionability
  - volunteer handoff clarity
  - low-pressure fit
  - sync understanding
- Pilot readiness gate for:
  - consent packet
  - teacher brief
  - test device
  - feedback form

## Pilot Principle

The classroom pilot should test whether English+ helps students return to a small task after frustration, and whether teachers or volunteers can act on the handoff evidence. It should not measure students by public ranking or raw score comparison.

## Still Needed Before a Real Classroom Pilot

- Prepare the actual consent/notice text for the school context.
- Choose the test device and install the latest debug APK.
- Prepare a one-page teacher brief.
- Prepare a short feedback form for students and staff.
