# Round 11 - Classroom Pilot Materials

This round turns the classroom pilot plan into practical materials that can be handed to teachers, students, volunteers, and the course team.

## Added

- `PilotMaterialsContract` with schema version `11`.
- Teacher brief:
  - English+ is a low-pressure learning and emotional-breakpoint support prototype.
  - It is not a ranking or test-first tool.
  - Teachers and volunteers should focus on evidence and the next small support action.
- Consent notice:
  - what data is used for
  - what data may be recorded
  - who can see it
  - how AI is used
  - opt-out path
- Feedback form:
  - student return willingness
  - task pressure
  - teacher actionability
  - report usefulness
  - volunteer handoff clarity
- Observation sheet:
  - student returned after support
  - teacher next action identified
  - handoff summary understandable
  - sync/report state understandable

## Pilot Use

Before a classroom or presentation pilot, the course team can turn these contract values into a one-page teacher brief, consent notice, Google Form, and observer checklist.

## Guardrail

The materials explicitly avoid public ranking, raw score comparison, and deficit framing. The pilot should evaluate whether English+ helps students return to a small task and whether staff can act on the handoff evidence.
